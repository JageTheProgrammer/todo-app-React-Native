Todo App Made with React-Native + Typescript + Expo + Node.js + MongoDB. 

Simple App where you can Add Things you need to do and markup em when that specified task has been done.

Multiple Colour Themes And Dark/Light Switch
Thanks for reviewing my project. <3



To start project firstly on backend folder in cmd install the packages, edit .env.example to .env and add your mongodb then start the dev server by command "npm run dev"
if it starts successfully then open frontend on cmd and install packages and then start the project with "npm run start" or "npx expo start".
